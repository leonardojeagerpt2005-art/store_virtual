# Sales Program

## Description

This program is a simple sales system made in Python.

The program lets the user register sales in a store.

The user writes:

* Product name
* Product price
* Quantity

Then the program calculates the total price.

## Strategy

The program uses functions to organize the code.

One function registers the sales.

Another function calculates the totals.

Another function shows the sales summary.

The program also uses a list to save all sales.

A loop is used to register many sales.

At the end, the program shows the total sales and the total money.

## Goal

The goal of this program is to practice Python basics.

For example:

* Functions
* Lists
* Loops
* User input
